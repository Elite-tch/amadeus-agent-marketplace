# Quick Start Guide

Get up and running with the Amadeus Agent Marketplace in minutes.

## Prerequisites

Before you begin, you'll need:
- A modern web browser (Chrome, Firefox, or Edge)
- The Amadeus Wallet Extension installed

## Step 1: Install Amadeus Wallet

1. Visit the [Chrome Web Store](https://chromewebstore.google.com/detail/amadeus-wallet/gigmkdnbhopbandngplohmilogilbkjn)
2. Click **Add to Chrome**
3. Follow the prompts to create a new wallet or import an existing one
4. **Save your seed phrase** in a secure location

## Step 2: Get AMA Tokens

To purchase agents or publish your own, you'll need AMA tokens.

**For Testnet:**
- Visit the [Amadeus Faucet](#) to get free testnet tokens
- Enter your wallet address
- Claim tokens (may take a few minutes)

**For Mainnet:**
- Purchase AMA from supported exchanges
- Send tokens to your wallet address

## Step 3: Connect Your Wallet

1. Navigate to [Amadeus Agent Marketplace](/)
2. Click **Connect Wallet** in the top right
3. Approve the connection in the wallet popup
4. Your wallet address and balance will appear

## Step 4: Browse Agents

1. Go to the **Explore** page
2. Browse available agents by category
3. Use the search bar to find specific agents
4. Click on an agent card to view details

## Step 5: Purchase an Agent

1. Find an agent you want to access
2. Click **Get for X AMA**
3. Review the transaction in the pop-up modal
4. Click **Confirm Purchase**
5. Approve the transaction in your wallet
6. Wait for blockchain confirmation
7. Access your agent in **My Agents**

## Step 6: Publish Your First Agent (Optional)

1. Ensure you have at least **1 AMA** for the publishing fee
2. Navigate to the **Publish** page
3. Fill in agent details (name, description, category)
4. Add your MCP server URL
5. Set pricing (free or paid)
6. Upload media (logo, demo video, screenshots)
7. Pay the 1 AMA publishing fee
8. Submit your agent for listing
